# NBCG
Digital Library for National Library of Montenegro "Đurđe Crnojević"

## Using
### dockers
- MinIO
- OpenSearch
- Nginx Proxy Manager
- PostgreSQL
### frontend
- Vue.js
### backend
- Express.js

# Quick start

```./init.sh```

```./run.sh```

## Start Developemnt
1. initalize

```./init.sh dev nbcg```

3. create / install dockers & npm packages

```nbcg create```

3. start dockers, frontend and backend in dev environment

```nbcg start```
