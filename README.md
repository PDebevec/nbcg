# NBCG
Digital Library for National Library of Montenegro "Đurđe Crnojević"

## Using
### dockers
- MinIO
- OpenSearch
- Nginx Proxy Manager
- PostgreSQL
### frontend
- Vue.js
- Quasar
### backend
- Express.js
- Prisma

# Quick start

```make qs```

## Start Development
1. initalize

```make init```

3. create / install docker containers & npm packages

```make create```

3. start dockers, frontend and backend in dev environment

```make start```
